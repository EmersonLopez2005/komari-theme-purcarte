## 开发

1.  **克隆项目**

    ```bash
    git clone https://github.com/Montia37/komari-theme-purcarte.git
    cd Komari-theme-purcarte
    ```

2.  **安装依赖**

    ```bash
    yarn install
    ```

3.  **运行开发服务器**

    ```bash
    yarn dev
    ```

4.  在浏览器中打开 `http://localhost:5173` (或 Vite 提示的其他端口)。
